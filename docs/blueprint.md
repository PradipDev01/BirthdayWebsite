# **App Name**: Tulip Wishes

## Core Features:

- Photo Gallery: Display a collection of beautifully framed photos of the birthday celebrant.
- Birthday Wish Submission: Allow other users to submit birthday wishes via a form on the site.
- Wish Display: Showcase the submitted birthday wishes in a beautifully designed section.
- Wish Storage: Store wishes from users in Firestore, so that they can be displayed on the page. Implement simple moderation features.
- Theme Personalization: Customize theme elements using AI. The AI will be presented with the page as a tool. If appropriate, it will make decisions about whether to introduce new colors or graphical elements, related to tulips.

## Style Guidelines:

- Primary color: Soft, muted rose (#E7DAD8) to evoke a sense of warmth and affection, reminiscent of tulip petals.
- Background color: Light blush (#F9F5F6), a very lightly saturated tint of the primary, to keep a focus on photography and user wishes without eye strain.
- Accent color: Dusty lavender (#A99CB0), analogous to rose, to provide visual interest.
- Headline font: 'Belleza' sans-serif for headings and short messages, imparting an artful feel. Body font: 'Alegreya' serif for user-submitted wishes.
- Use delicate, custom icons inspired by tulips and related floral elements.
- Design a soothing, minimalist layout with ample white space to highlight photos and wishes.
- Incorporate gentle animations, such as subtle fade-ins, for a soft and elegant user experience.
